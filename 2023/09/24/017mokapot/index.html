摩卡壶改进方案一则

发表于 2023-09-24 更新于 2024-02-20 分类于 设计 ， 产品 阅读次数： Valine：
本文字数： 625 阅读时长 ≈ 1 分钟

给自己练手的虚拟项目……

原产品分析

摩卡壶是一种萃取意式浓缩咖啡基底的工具，分为上中下结构，下座是盛水的水槽，中间的粉槽盛放咖啡粉，上座盛放萃取后的咖啡液。它的原理是通过加热下壶中的水变成蒸气，利用蒸气的压力将水推升至导管进入粉槽而萃取咖啡液，再继续通过导管推升到上壶聚合流出。

在最近半年使用摩卡壶的过程中，我发现传统的摩卡壶仍然有几大不足之处。

  1. 壶体由铝制成，易氧化，且不利于清洗；新式摩卡壶可能由不锈钢制成，尽管避免生锈，但依旧有不便清洗等问题。
  2. 对新手来说，掌握火候可能面临问题，并导致做出的咖啡味道不尽人意。如使用不慎，高温液体还可能喷溅，对人体造成伤害。
  3. 压粉力度不好掌握，需要额外的布粉器辅助，且粉槽在清洗的时候很难取出。
  4. 摩卡壶原本设计用于明火，虽然现在可以用电陶炉等加热，但不足以适应部分人群对城市快节奏生活的需求。
  5. 制作完成后下壶体温度极高，不便与上壶体分离，可能导致过萃。

设计思路

  1. 提取原有的壶体八角结构，但打磨至更圆润，在现代室内装修环境中更协调。
  2. 采用陶瓷制作壶体，便于清洗，不易生锈，但是需考虑工艺、造价问题。
  3. 用压粉模块替代外置布粉器，并用渐变色直观指示压粉深度。
  4. 集成加热、恒温模块，避免使用炉灶、电陶炉的麻烦，随时随地享用咖啡。

说明

  1. 由于建模能力所限，八角结构的提取不够完美，存在接缝线等问题；壶盖与壶体的连接也没有建立模型。
  2. 压粉模块并没有设计完整的结构和原理，更偏向概念性。
  3. 陶瓷材质的渲染及打光等有待改进。
